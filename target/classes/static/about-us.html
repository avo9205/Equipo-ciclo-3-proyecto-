Historia del proyecto

En el contexto global donde los servicios web son cada vez mas utilizados, hace fundamental y necesario la formacion y preparacion de nuevos desarrolladores que permitan solventar la creciente demanda en el desarrollo de estos servicios.
Por tal motivo y a traves de mision Tic 2022 hemos desarrollado un servicio web en el que simulamos la operacion de una empresa de ropa donde se registran los movimientos financieros ,las ventas realizadas por cada empleado y la empresa a la que pertenece cada uno.
Para tal fin aplicamos el protocolo API REST para elabororar dicho servicio utilizando tecnologias como spring boot para el desarrollo del back end y enlazandolo a una base de datos externa con SupaBase y posteriormente utilizando thymeleaf para el desarollo del front end aplicando plantillas por medio de boostrap. Ademas utilizamos Apis como Auth0 para permitir el registro de los usuarios con sus correspondientes contraseñas y con heroku montamos nuestro servicio a la web.
